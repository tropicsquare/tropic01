## Datasheet and API

> [!IMPORTANT]
> TROPIC01 is a software-defined product. Chips ship from manufacturing with **initial** firmware versions for both application and SPECT cores.
>
> Update to the latest firmware using [libtropic](https://github.com/tropicsquare/libtropic) for full functionality.
