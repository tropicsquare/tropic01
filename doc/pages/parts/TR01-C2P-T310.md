# TR01-C2P-T310

Complete reference for this TROPIC01 part number.

## Datasheet and API

> [!IMPORTANT]
> TROPIC01 is a software-defined product. Chips ship from manufacturing with **initial** firmware versions for both application and SPECT cores.
>
> Update to the latest firmware using [libtropic](https://github.com/tropicsquare/libtropic) for full functionality.

Parts are shipped with:

| Application FW version | SPECT FW version | Datasheet version | API version |

| 2.0.0 | 1.0.0 | [A.11 (PDF)](../doc/datasheet/TROPIC01_datasheet_A11.pdf)| [1.4.0 (PDF)](../doc/datasheet/TROPIC01_User_API_1v4v0.pdf) |
