# TR01-C2P-T310

This page collects all the information related to particular TROPIC01 part number.

## Datasheet and API

> [!IMPORTANT]
> TROPIC01 is a software-defined product whose functionality depends on the loaded firmware. From manufacturing the chip ships with **initial** firmware versions for both the application core and SPECT core, which determine its default behavior. 
> 
> Follow instructions in [libtropic](https://github.com/tropicsquare/libtropic) and update your chip to latest fw versions.
